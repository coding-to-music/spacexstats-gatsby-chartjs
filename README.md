## Installation

### Prereqs

- Node >= v8.0.0
- Yarn >= v1.13.0

`npm install --global yarn`

### Setup instructions

1. In the project directory, run the `yarn` command.
2. Run `yarn start` OR `yarn start --open` to skip step 3
3. Open `http://localhost:8000/`.
4. ...
5. Profit!
